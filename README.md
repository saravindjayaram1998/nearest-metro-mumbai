# nearest-metro-mumbai

An Android app that finds the nearest Mumbai Metro station from your current location and opens Google Maps for the route.

## Repo structure
- `android-app/` Android Studio project (Kotlin + Jetpack Compose)
- `docs/` GitHub Pages project blog
- `assets/` screenshots, gifs
- `data/` station dataset sources
