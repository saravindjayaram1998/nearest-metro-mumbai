# Nearest Metro Mumbai

This project finds the nearest Mumbai Metro station from your current location and opens Google Maps for directions.

## What you'll find here
- App build notes
- Architecture and design decisions
- Dataset info (stations)
- Privacy policy (needed for Play Store)

## MVP checklist
- [ ] Ask for location permission
- [ ] Get current location
- [ ] Load station list (JSON)
- [ ] Compute nearest station
- [ ] Open route in Google Maps
